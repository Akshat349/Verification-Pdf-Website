SYSTEM OVERVIEW

The application acts as a central hub for reporting the status of production or operational elements. It uses the 4M Management philosophy:

  • Man: Personnel and labor status.
  • Machine: Equipment and infrastructure health.
  • Material: Raw materials or inventory availability.
  • Method: Processes and standard operating procedures.

2. Structural Breakdown

A. General Info (Navigation)

Allows users to understand the current context and review history via the Description and View Report modules.

B. Report Submission (Action Center)

Categorizes reporting into two logical paths:

Submit No-Change: A "quick-confirm" feature for standard operations.

Submit Change: An "exception-reporting" feature to detail deviations or breakdowns.

C. Tools & Status (Information)

Provides instructional tooltips and live system metadata (v1.0).
